# Oversized-raspberry
The oversized raspberry is a rp2354 based board in the form factor of the arduino uno.
-----------COMPATIBILITY NOTE-----------
THIS IS NOT A DERIVATIVE WORK OF THE ARDUINO UNO.

This is a 3.3v system, not a 5v system and has no buck converter for 12v, so while this is similar to the uno in shape, this is not intercompatabile in addition to the fact that that pin mapping was not designed for Arduino inter compatibility.
---------

The oversized raspberry is a rp2354 based board with a similar design to the arduino uno. It is a 3.3v system utilizing all of the anologe pins. This board is very simple and is self sourcable from lcsc or your prefered distributor.
